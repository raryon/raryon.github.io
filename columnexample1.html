Area 1

The normal placement of content on a web page is to place it after the previous content, flowing from the left to the right and then down to the next line.

Areas can be created using HTML div tags with ids or it can be created with the semantic HTML tags such as article, section, aside.

Block display style areas begin on a new line and often have extra space above and below.

Area 2

Multicolumn web pages can be created using various techniques. The four most common techniques are:

  1. Float and clear CSS properties
  2. Flexbox CSS property
  3. CSS frameworks
  4. CSS grid
