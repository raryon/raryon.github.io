HTML Block elements

The basic layout of a web page built with HTML is to use blocks. The paragraph element is the most basic block element. When you use the html <p> tag the following block will begin on a new line. When you press the enter key in your content, you will not create a new paragraph. You must first end the current paragraph with the end paragraph tag and start the next paragraph with a new paragraph tag.

When you begin a second paragraph with a new paragraph tag, there will be an extra blank line between the two paragraphs.
Another way to start your text on a new line is to add a br tag. The new content will start on a new line, but no extra blank line will appear before this content.

Heading elements

Use the h1 through h6 tag to add headings elements to your web page. The content will be bold and show various font sizes. The lower the number 1 - 6 the larger the font size. These are also block elements, therefore the content will begin on a new line.

List elements

  • Lists are contained inside of ul or ol tags.
  • Unordered lists use the ul tag.
  • Ordered lists use the ol tag.
  • Each list item is contained in a li element.
