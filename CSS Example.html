Using CSS to add Style

CSS allows a web developer a method to format the web page. Some of the styles that can be set with CSS is color, text font, size of elements and position of elements.

Three methods to add CSS to an HTML document.

  • Inline
  • Internal
  • External

This paragraph uses an inline style. The style attribute is added to the p tag to give the paragraph a black border and blue text.

Internal styles are placed inside of a begin and end style tag in the head section of the web page. To identify the html element that is being styled a selector must be used. The CSS rules follow the selector and are placed inside of curly braces. The selector identifies the HTML element that uses the defined CSS rules.

External styles are placed in a different file. Similar to internal styles, the html element that is being styled is identified by a selector and the CSS rules follow the selector and are placed inside of curly braces. A link statement is placed in the head section of the html that indicates the name of the CSS file.

Selectors

You will notice that selectors can be of various types. Some selectors are the name of the HTML element. In this page the internal CSS uses the header selector to create a maroon background color for the header area of the page.

Other selectors might refer to an ID attribute value, these will have a Hash tag in front of their name in the CSS code. Notice the div tags often use an id. Look in the CSS code in the head section to find the id selector with a hash tag.

Another type of selector uses a class attribute value, and will have a dot in front of it in the internal style area. Looking at the code you will see a span area with a class value of special. Notice that the corresponding CSS selector is .special.
